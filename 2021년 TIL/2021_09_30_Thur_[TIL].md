## <React.Fragment></React.Fragment>

The React.Fragment component lets you return multiple elements in a render() method without creating an additional DOM element:

```jsx
render() {
  return (
    <React.Fragment>
      Some text.
      <h2>A heading</h2>
    </React.Fragment>);
}
```

You can also use it with the shorthand `<></>` syntax.
